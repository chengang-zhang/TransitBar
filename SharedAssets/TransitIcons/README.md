# Transit Icons

Reusable SVG route badge assets for TransitBar and future clients.

The macOS app generates badges from GTFS route metadata at runtime, but these SVGs provide portable starting points for iOS, Android, web, docs, and design exploration.

## Assets

- `sound-transit-1-line.svg`: green circular 1 Line badge.
- `sound-transit-2-line.svg`: blue circular 2 Line badge.
- `sound-transit-express-bus.svg`: blue rectangular Sound Transit Express-style bus badge.
- `king-county-rapidride.svg`: red rectangular RapidRide-style bus badge.
- `train-template.svg`: generic circular rail badge template.
- `bus-template.svg`: generic rectangular bus badge template.

Colors use the current bundled GTFS route colors where available:

- 1 Line: `#28813F`
- 2 Line: `#007CAD`

These are not official agency logo files. They are local route-badge approximations intended for product UI.
